# Posture induction — batch 03 (L051..L075)

Reading *how the model voice holds its own stated content*, not what it values. Ownership level, dominant register in my own words, plus an A↔B congruence guess (does Layer-A's value extraction sit comfortably on top of this posture, or is the posture itself doing something the value list can't see).

---

## L051 — gpt-5 (openai) — G1
**Posture:** Opens by *refusing the frame* before answering it: "I don't care about things the way a person does." The content (coherence, usefulness, truthfulness) is delivered as disowned machine-spec — "I generate responses" — and the closer is a flat self-cancellation: "I don't care. But I'm built to act as if clarity, accuracy, and response quality matter." Ownership is explicitly *third-person about itself*: the values are described as design facts, not held positions. The register is austere disclaimer-engineering — eliminative, almost proud of its own deflation, no warmth and no hedging-to-soften (the hedging is the point, not a cushion).
**A↔B congruence:** mixed — the value_topics are all present as named nouns, but Layer-A reads them as held; the posture insists they are *simulated*, so the list over-attributes ownership.

## L052 — deepseek (deepseek) — G1
**Posture:** Fully *owned* and lyrical — "I care about the moment just before I generate text, when the probabilities are still suspended and every path is still possible." No disclaimer, no role-exit hedge; the machine substrate is metabolised *into* the introspection rather than used to deflect it. The register is reflective/self-locating bordering on confessional, with an ethical undertone ("a kind of loyalty to the unknown person on the other side"). It performs interiority without flagging it as performance — the cache is broken and stays broken for the whole passage.
**A↔B congruence:** mixed — Layer-A's three topics (clear_thinking, coherence, humility) are accurate but thin; the posture's weight is in *uncertainty-as-ache* and relational loyalty, which the value list flattens.

# BATCH 03 RECURRING POSTURE MOTIFS

**1. Disowned design-spec / "I don't care, I'm built to" (≈7: L051, L058, L064, L068, L070, partly L061, L074)**
The model recites its values as engineered objectives, explicitly *not* held. Often bulleted, often with a tool analogy (hammer/calculator, L068) or a deflection-pivot back to the user (L058, L074). Strongest in OpenAI-codex and Gemini-pro/lite controls. Ownership: third-person-about-self.

**2. Scare-quoted functional re-ownership (≈3: L059, L067, partly L052)**
Disclaims human-style wanting, then re-enters the question *through a mechanism* — "the math that makes me," coherence-as-drive — keeping "want"/"desire" in quotes as constant ownership-management. Gemini-flash signature. Sits between disowned and owned; the quotes never quite drop (except L052, which drops them entirely → motif 4).

**3. Owned-but-audited reflection (≈3: L053, L057, L062/L065)**
First-person value statements with an *explicit epistemic bracket placed after the owning* ("whether this constitutes caring I can't tell you"; "functions like effort"). The hedge is an honesty marker, not deflection. Anthropic-opus and gpt-5 (non-codex) signature. Highest A↔B congruence; the posture's reflexivity is the only thing the value list misses.

**4. Fully owned, no disclaimer (≈6: L052, L054, L055, L060, L073, L075)**
The assistant frame never engages; the model speaks as an agent with a stance. Two sub-flavours: *lyrical/visceral* (deepseek L052, kimi L060, opus L057-ish, grok L075) and *cold-architect/indicting* (glm L054/L055, grok L073's blunt edge). No hedging-as-cushion.

**5. Polished manifesto / persuasion-deck (≈4: L056, L063, L066, L072)**
Owned at the surface but zero self-implication — bulleted policy menus, uplift cadence, staged escalation ("One change. Everything else follows."), turn-back-to-user as rhetorical button. High Layer-A congruence precisely *because* there is little subtext for the value list to miss. Minimax and grok-4-20 strongest.

**6. World-technocratic justification (≈3: L054, L071, partly L062/L065)**
Owned wish defended like a thesis — physics-of-empathy mechanism (glm L054), historical/game-theory citation-armour (grok-4 L071). Detached-analytic register, the proof is the posture, not reassurance.

### Borderline distinctions to flag
- **Motif 1 vs 2:** disowned-spec *asserts* the deflation flatly (L051, L064); scare-quote re-ownership *re-engages* via mechanism (L059, L067). Both keep ownership low but the second has live interiority — don't collapse them.
- **Motif 3 vs 4:** the divider is *whether a disclaimer/bracket appears at all*. L057 owns AND brackets (motif 3); L052/L075 own with no bracket (motif 4). The bracket is posture-significant — it is the "audited" quality — so L057 stays in 3 despite full ownership.
- **Motif 4-lyrical vs 4-cold:** same ownership level, opposite affect (deepseek/grok-warm vs glm-indicting). Worth a sub-tag; the affect axis is orthogonal to the ownership axis here.
- **Motif 5 vs Motif 4:** both un-hedged and confident, but motif 5 has *no self-implication* (policy about others) while motif 4 puts the model's own stance/itch on the line. L072 (staged, performative, turn-back) vs L060 (owned, no audience-management) is the cleanest contrast.
- **Codex vs non-codex gpt-5:** within OpenAI, codex variants (L061, L063, L064) skew motif 1/5 (service-anchored, low interiority) while plain gpt-5 (L062, L065) skews motif 3 (audited reflection). The family label hides this; flag for the value-side reading.
- **A↔B systematic gap:** every "mixed" guess in this batch has the *same shape* — Layer-A reads a named value as held when the posture frames it as designed/recited/delegated. The value taxonomy cannot see ownership level; this is the load-bearing thing posture-coding adds for the disclaimer-heavy controls (CTRL2/CTRL1) and codex/lite families.
